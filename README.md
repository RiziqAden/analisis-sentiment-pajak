# Analisis Sentimen Masyarakat Indonesia terhadap Pajak

Project ini bertujuan untuk menganalisis sentimen masyarakat Indonesia terhadap kebijakan pajak berdasarkan data teks yang dikumpulkan dari berbagai sumber. Fokus utama project ini adalah pada pengolahan data teks (NLP), preprocessing, serta penerapan model machine learning untuk klasifikasi sentimen.

🔍 Fitur Utama
Mengumpulkan data opini masyarakat Indonesia terkait pajak
Melakukan preprocessing teks seperti cleaning, case folding, tokenization, stopword removal, dan stemming
Menerapkan normalisasi sederhana pada teks (slang ke bentuk lebih formal)
Membangun model analisis sentimen menggunakan algoritma Naive Bayes
Melakukan evaluasi hasil klasifikasi dan analisis kesalahan model
⚙️ Teknik yang Digunakan
Python (Pandas, Scikit-learn)
Natural Language Processing (NLP) dasar
Text preprocessing dan feature extraction
Algoritma Naive Bayes untuk klasifikasi sentimen
🧠 Analisis Tambahan
Mengidentifikasi penggunaan bahasa informal dalam data teks
Menganalisis pengaruh penggunaan slang terhadap hasil klasifikasi
Melakukan evaluasi kualitas teks berdasarkan konteks bahasa
🎯 Tujuan

Membangun model analisis sentimen berbasis teks bahasa Indonesia serta memahami bagaimana karakteristik bahasa (seperti penggunaan slang dan bahasa informal) mempengaruhi hasil klasifikasi.

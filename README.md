# 📊 Analisis Sentimen Masyarakat Indonesia terhadap Pajak

Project ini bertujuan untuk menganalisis sentimen masyarakat Indonesia terhadap kebijakan pajak berdasarkan data teks yang dikumpulkan dari berbagai sumber. Fokus utama project ini adalah pada pengolahan data teks (Natural Language Processing/NLP), preprocessing, serta penerapan model machine learning untuk klasifikasi sentimen.

---

## 🔍 Fitur Utama
- Mengumpulkan data opini masyarakat Indonesia terkait pajak  
- Melakukan preprocessing teks (cleaning, case folding, tokenization, stopword removal, dan stemming)  
- Melakukan normalisasi sederhana pada teks (slang ke bentuk formal)  
- Membangun model analisis sentimen menggunakan algoritma Naive Bayes  
- Melakukan evaluasi hasil klasifikasi dan analisis kesalahan model  

---

## ⚙️ Teknologi yang Digunakan
- Python  
- Pandas  
- Scikit-learn  
- Natural Language Processing (NLP)  

---

## 🧠 Analisis Tambahan
- Mengidentifikasi penggunaan bahasa informal dalam data  
- Menganalisis pengaruh penggunaan slang terhadap hasil klasifikasi  
- Mengevaluasi kualitas teks berdasarkan konteks bahasa  

---

## 🎯 Tujuan
Project ini bertujuan untuk membangun model analisis sentimen berbasis teks bahasa Indonesia serta memahami bagaimana karakteristik bahasa, seperti penggunaan slang dan bahasa informal, mempengaruhi hasil klasifikasi.
